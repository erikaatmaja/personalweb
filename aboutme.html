Hobbies : Drawing, Crafting

Saya adalah seorang mahasiswa jurusan Informatika angkatan 2018 di Universitas Multimedia Nusantara.
Saya membuat personal website ini untuk memperkenalkan diri saya sebagai seorang artist.
Melalui sosial media saya membagikan karya ilustrasi yang saya ciptakan.
Pada awalnya saya hanya bermodalkan kertas kosong dan alat mewarnai dengan menggunakan cat air (saya melakukan teknik watercolour),
lalu pada akhirnya saya mulai mencoba hal baru.
Media yang saya gunakan saat ini berdasarkan pada digital, atau akrab disebut sebagai ilustrasi digital.

Dibalik itu, saya membuka jasa ilustrasi (commission) sebagai uang jajan tambahan, hehe!
